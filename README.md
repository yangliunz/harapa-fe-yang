# Code Challenge Project

## Tech stack

```
React
TypeScript
Vite
Tailwind CSS
React Query
Vitest
React Testing Library
```

## Installation

`git clone https://github.com/yangliunz/harapa-fe-yang`
`cd harapa-fe-yang`
`pnpm install`
OR
`npm install` / `yarn install`

## Development

`npm run dev`

Visit http://localhost:5173 in your browser.

## Testing

`npm test`
